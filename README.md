TinDog Project

Overview

TinDog is a responsive landing page inspired by Tinder, designed specifically for dog owners looking to connect their pets with other dogs.

The project was built to practice modern web development techniques using HTML, CSS, and Bootstrap while creating an attractive and mobile-friendly user interface.

---

Features

- Responsive design for desktop and mobile devices
- Modern landing page layout
- Bootstrap components and grid system
- Call-to-action sections
- Testimonials section
- Pricing section
- Custom styling and branding

---

Technologies Used

- HTML5
- CSS3
- Bootstrap 5

---

Project Objectives

This project was developed to strengthen skills in:

- Frontend Web Development
- Responsive Web Design
- Bootstrap Framework
- CSS Styling
- User Interface Design
